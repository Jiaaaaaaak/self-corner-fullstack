# 專家回饋與深度分析規範 (Feedback Page Spec)

`/feedback` 頁面在練習結束後提供專業的量化指標與質性建議。

## 1. 佈局結構
*   **頂部**: 標題與動作列 (包含「重試一次」與「分享練習」按鈕)。
*   **主內容**: 雙欄佈局 (Left: 400px, Right: fill)。

## 2. 左欄：量化指標 (Quantitative Metrics)

### 2.1 五力雷達圖 (Radar Chart)
*   **座標軸**: 自我覺察、自我管理、社會覺察、人際關係、決策能力。
*   **數據顯示**:
    - **本次得分**: `#E07A5F` 實線，搭配 20% 透明度填充，頂點有 6px 圓點。
    - **平均得分**: `#E5E2D9` 虛線，搭配 10% 透明度填充。
*   **圖表背景**: `#F8F7F4`。

### 2.2 評分卡片 (Score Cards)
*   **綜合評分**: 顯示等級（如: A+），文字顏色 `#81B29A`。
*   **成長幅度**: 顯示百分比（如: +12%），文字顏色 `#E07A5F`，配有 `arrow-up-right` 圖示。

### 2.3 學生情緒流動圖 (Student Emotion Flow Chart)
*   **圖表類型**: 平滑折線圖 (Smooth Line Chart)。
*   **座標軸定義**:
    - **縱向 (Y-axis)**: 情緒類別（由下而上：抗拒/防衛 `#B54A4A` → 焦慮/退縮 `#F2CC8F` → 平靜/中性 `#A09C94` → 一致/開放 `#81B29A`）。
    - **橫向 (X-axis)**: 對話時間/回合數。
*   **視覺設計**:
    - **線條**: `#E07A5F` (Terracotta)，粗細 3px，具備節點圓點。
    - **陰影**: 線條下方具備淺色漸層填充 (Area fill)。
    - **互動**: 點擊節點可快速跳轉至右側「對話逐字稿」的對應位置。

## 3. 右欄：質性分析 (Qualitative Analysis)

### 3.1 KIST 對話卡 (KIST Cards)
*   顯示本次對話中 AI 教練判斷最相關的 KIST 12 張對話卡。
*   卡片應包含標題、對應精神與精美圖示。

### 3.2 專家建議 (Expert Suggestions)

*   採用分頁標籤切換「專家建議」與「對話逐字稿」。
*   **建議區塊結構**:
    - **✅ 亮點肯定**: 綠色標題 (`#81B29A`)。
    - **🔍 盲點發現**: 橘色標題 (`#E07A5F`)。
    - **原本可以這樣說 (Quote Block)**: 
        - 背景 `#F8F7F4`，左側 3px `#E07A5F` 邊框。
        - 包含 `quote` 圖示與斜體建議文字。
    - **🎯 行動建議**: 金色標題 (`#F2CC8F`)。

## 4. 交互規範
*   **分享練習**: 點擊後應彈出分享介面或複製分享連結。
*   **重試一次**: 回到對話空間並重置狀態。
