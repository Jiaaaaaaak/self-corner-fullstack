# 文案與溝通語調規範 (UX Writing & Tone of Voice)

本文件定義 SELf-corner 的語言風格，以共情式諮商語調為核心，打造溫暖、專業且具支持性的溝通體驗。

## 1. 語調核心 (Core Tone)

*   **共情 (Empathetic)**: 以老師與家長的視角出發，理解孩子在每一刻的情緒與需求，用溫暖的語句陪伴使用者。
*   **專業 (Authoritative)**: 以薩提爾模式與 SEL 框架為依據，確保每一句文案都有理論支撐。
*   **支持 (Supportive)**: 以正向回饋為導向，在每個互動環節中給予肯定與鼓勵。

## 2. 寫作準則 (Writing Rules)

### 2.1 頁面標語 (Slogans)
*   **DO**: "準備好進入練習了嗎？" (溫暖邀請)
*   **DON'T**: "開始測試" (具備壓力感)

### 2.2 錯誤訊息 (Errors)
*   **DO**: "哎呀，連線似乎有點不穩，請稍後再試一次吧！" (共擔責任)
*   **DON'T**: "ERROR 500: Server disconnected" (冰冷技術術語)

### 2.3 專家建議結構 (Feedback Structure)
*   **流程**: `肯定亮點` → `發現盲點` → `行動導向建議`。
*   **範例**: 「你在這次對話中展現了很好的傾聽能力！不過在回應情緒的部分，可以試著多用『我注意到...』的句型來表達同理心。」

## 3. 國際化與術語標準 (i18n)
