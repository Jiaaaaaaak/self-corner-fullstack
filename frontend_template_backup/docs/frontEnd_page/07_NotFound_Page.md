# 404 錯誤引導設計規範 (NotFound Page Spec)

當用戶走錯路徑時，提供溫暖且有趣的導引回航。

## 1. 視覺設計 (Visual Design)
*   **背景**: 使用 `#FAF9F6` 到 `#F0EDE6` 的線性漸層。
*   **核心插圖**: 
    - 迷路學生 Emoji (🧑‍🎓, 72px)。
    - 上方有三個橘色問號 (`? ? ?`, `#E07A5F`)。
*   **錯誤代碼**: 大字體 "404" (120px, `#E5E2D9`, 負 4px 字距)。

## 2. 交互與反饋
*   **Headline**: "哎呀，這個教室沒有人呢" (24px, Bold)。
*   **Body**: "看來你走錯了路。不過沒關係，我們一起回大廳看看吧！" (16px, 顏色 `#706C61`)。
*   **自動跳轉**: 
    - 顯示 "5 秒後自動返回首頁" 文字。
    - 下方有進度條 (背景 `#E5E2D9`, 填充 `#E07A5F`)。

## 3. 行動按鈕 (CTA)
*   **主要按鈕**: "帶我回首頁" (背景 `#E07A5F`, 搭配 `arrow-left` 圖示, 具備陰影 `shadow-lg`)。
*   **次要連結**: "或者去對話練習看看 →" (文字顏色 `#E07A5F`)。

## 4. 裝飾元素
*   背景具備隨機分佈的粉筆裝飾線條與小圓點，營造教室黑板的感覺，透明度設定為 `0.08` 到 `0.15`。
